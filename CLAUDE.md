# Laden-IT-Lab

## Über mich
Ich bin Anfänger und lerne im Rahmen meiner Ausbildung zum Fachinformatiker
Systemintegration. Bitte erkläre mir in diesem Projekt immer, was du tust
(welchen Befehl, welche Datei, warum), und gehe dabei in kleinen,
nachvollziehbaren Schritten vor. Keine großen Sprünge oder viele
Änderungen auf einmal, ohne sie kurz zu erläutern.

## Projektziel
Ein kleines Kassensystem (Point-of-Sale) als Lernprojekt.

## Tech-Stack
- Sprache: Python
- Backend-Framework: FastAPI
- Datenbank: SQLite

## Geplanter Betrieb
Das System soll später auf echten Servern betrieben werden, unter anderem:
- Linux-Server
- Windows Server

Dabei sollen folgende Themen mit einfließen (auch als Lernfelder für die
Ausbildung):
- Nutzerverwaltung (Benutzer, Rollen, Rechte)
- Backups
- Monitoring
- Helpdesk / Support-Prozesse
